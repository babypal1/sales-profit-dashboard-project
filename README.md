# Sales & Profit Analysis Dashboard

## 📌 Project Overview
This project demonstrates end-to-end data analysis:
- Data cleaning using Python (Pandas)
- Data storage using SQL
- Dashboard creation using Power BI

## 📂 Files Included
- raw_data.xlsx
- cleaned_data.xlsx
- data_cleaning.py
- dashboard.pbix

## 🚀 Tools Used
- Python
- Pandas
- SQL
- Power BI

## 📊 Key Insights
- North region has highest profit
- Printer has highest sales
- Laptop gives better profit margin
